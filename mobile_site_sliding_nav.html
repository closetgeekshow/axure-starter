Mobile site with

sliding nav


This page uses a Hidden dynamic panel (yellow mask) that sticks to the browser (Pin to browser) to set up sliding navigation.


The header is a pinned dynamic panel that contains a menu icon that shows the sliding nav.


The Loaded event on the nav panel moves it to 0,0 and brings it to the front so feel free to move it out of the way.


As you add pages to your prototype, consider creating a Component for the navigation.
